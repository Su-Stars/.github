## 프로젝트 소개

수영장 검색 & 기록 서비스

수스타는 사용자가 원하는 지역의 수영장을 쉽게 검색하고, 이용 후기 및 수영 기록을 관리할 수 있는 서비스입니다.

### 🔍 지역 필터링
